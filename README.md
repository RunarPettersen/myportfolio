# Portfolio (Runar Pettersen)

This repository contains my portfolio website, built as a static site with HTML, CSS, and JavaScript (ES Modules). The site presents selected school projects and improvements, with dedicated project articles and links to live demos, repositories, READMEs, and relevant commits.

## Live site

[Netlify](https://runarportfolio.netlify.app/)

## Repository

[Github](https://github.com/RunarPettersen/myportfolio)

## What’s included

- Static multi-page portfolio (no framework)
- Project article pages for:
  - Howling (CSS Frameworks)
  - Chopping Mall (JavaScript Frameworks)
  - Rusty Relics (Semester Project 2)
- Sketch / Finished mode toggle (starts in Sketch mode each visit)
- “Copy share link” button on project articles
- Responsive layout
- Social links in footer

## Tech

- HTML5
- CSS
- JavaScript (ES Modules)
- Google Fonts (Inter + Patrick Hand)
- WebP images

## Running locally

Because this is a static site, you can run it with any local web server.

Option A: VS Code Live Server
1. Open the project in VS Code
2. Right-click index.html and choose “Open with Live Server”
